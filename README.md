# snosser

```python
git clone https://github.com/vert5x-git/snoser
```
# Затем заходим в нужную папку
```python
cd snoser
```
# Устанавливаем зависимости 
```python
pip install requirements.txt
```
# Запускаем сносер
```python
python3 snos.py
```
